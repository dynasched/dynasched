Dynamic Scheduler (DynaSched) |dsched_ver|
==========================================

The project is formally referred to in documentation by "DynaSched", and
the GitHub repository is ``dynasched``.

.. note:: For brevity, all of the internal API symbols, environment
          variables, MCA frameworks, and CLI executables all use the
          abbreviated ``dsched`` for convenience.

Documentation locations
=======================

This documentation can be found in the following locations:

* On the web: https://dynasched.readthedocs.io
* In the tarball: ``docs/_build/html/index.html``
* Installed: ``$prefix/share/doc/dynasched/html/index.html``
